# Grocery List

## Overview

This is a simple interactive grocery list.
You can add products, remove and filter them to the list.

### Concepts developed

- DOM's elements manipulation (wuth events delegation)
- Array methods chaining

### Concepts

- HTML
- CSS
- JAVASCRIPT
- BOOTSTRAP (CDN)
- FONTAWESOME (script)
